# Demo Assets Folder

This folder contains assets for the Zypher AI demo.

## Required Images

Place your demo images in this folder with the following names:

### Logo Images (Demo Outputs)
- `demo-logo-1.png` - Modern tech startup logo example
- `demo-logo-2.png` - Coffee shop logo example  
- `demo-logo-3.png` - Fitness brand logo example
- `demo-logo-4.png` - Law firm logo example
- `demo-logo-default.png` - Default/fallback logo example

### Branding
- `zypher.jpeg` - Your Zypher AI logo/icon (appears in header and sidebar)

## Image Specifications

**Demo Logo Images:**
- Recommended size: 1024x1024px
- Format: PNG (supports transparency)
- File size: Keep under 500KB for faster loading

**Zypher Logo:**
- Recommended size: 512x512px or larger
- Format: JPEG or PNG
- Should be square for best display

## Notes

- All image references will gracefully fall back to placeholders if files are missing
- You can use any logo images you've created previously
- For GitHub Pages, ensure all images are optimized for web delivery
- The demo will work without images (showing placeholder graphics instead)

## Adding Your Own Images

1. Save your logo examples as PNG files
2. Rename them according to the list above
3. Place them in this `assets/` folder
4. The demo will automatically load and display them

That's it! The demo is ready to showcase your work.
